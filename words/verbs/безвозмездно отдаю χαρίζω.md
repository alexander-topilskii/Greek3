# База
дарю / безвозмездно отдаю : χάρισα - χαρίζω - θα χαρίσω

# Тип
Τύπος Α

# Формы
χάρισα - я подарил / отдал безвозмездно
χαρίζω - я дарю / безвозмездно отдаю
χαρίζεις - ты даришь
χαρίζει - он/она дарит
χαρίζουμε - мы дарим
χαρίζετε - вы дарите
χαρίζουν - они дарят
θα χαρίσω - я подарю / отдам безвозмездно

# Контекст
- **Της χάρισα το παλιό μου αυτοκίνητο.** — Я безвозмездно отдал ей свою старую машину.
- **Χαρίζω τα παλιά μου ρούχα.** — Я отдаю свою старую одежду (бесплатно).
