# База
хочу есть : πείνασα - πεινάω - θα πεινάσω

# Тип
Τύπος Β1 + Β2

# Формы
πείνασα - я захотел есть
πείνασα - я захотела есть
πείνασες - ты захотел есть
πείνασε - он захотел есть
πείνασε - она захотела есть
πείνασε - оно захотело есть
πεινάσαμε - мы захотели есть
πεινάσατε - вы захотели есть
πείνασαν - они захотели есть

πεινάω - я хочу есть
πεινάς - ты хочешь есть
πεινά - он хочет есть
πεινά - она хочет есть
πεινά - оно хочет есть
πεινάμε - мы хотим есть
πεινάτε - вы хотите есть
πεινάνε - они хотят есть

θα πεινάσω - я захочу есть
θα πεινάσεις - ты захочешь есть
θα πεινάσει - он захочет есть
θα πεινάσει - она захочет есть
θα πεινάσει - оно захочет есть
θα πεινάσουμε - мы захотим есть
θα πεινάσετε - вы захотите есть
θα πεινάσουν - они захотят есть
