# База
предлагаю : πρότεινα - προτείνω - θα προτείνω

# Тип
Τύπος Α

# Формы
πρότεινα - я предложил(а)
προτείνω - я предлагаю
προτείνεις - ты предлагаешь
προτείνουμε - мы предлагаем
θα προτείνω - я предложу

# Контекст
- **Προτείνω μια περιπέτεια.** — Предлагаю приключенческий фильм.
- **Προτείνω να πάμε στη συναυλία.** — Предлагаю пойти на концерт.
