# База
веду / записываю : κράτησα - κρατάω - θα κρατήσω

# Тип
Τύπος Α

# Формы
κρατάω - я веду / держу / записываю
κρατάτε - вы ведите / записываете (вежл.)
κρατά - он/она ведёт / держит

# Контекст
- **Κρατάω σημειώσεις στο μάθημα.** — Веду заметки на уроке.
- **Κρατάτε σύντομες σημειώσεις.** — Ведите краткие заметки.
