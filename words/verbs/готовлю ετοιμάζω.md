# База
готовлю/собираю : ετοίμασα - ετοιμάζω - θα ετοιμάσω

# Тип
Τύπος Α

# Формы
ετοίμασα - я подготовил
ετοίμασα - я подготовила
ετοίμασες - ты подготовил
ετοίμασε - он подготовил
ετοίμασε - она подготовила
ετοίμασε - оно подготовило
ετοιμάσαμε - мы подготовили
ετοιμάσατε - вы подготовили
ετοίμασαν - они подготовили

ετοιμάζω - я готовлю / собираю
ετοιμάζεις - ты готовишь / собираешь
ετοιμάζει - он готовит / собирает
ετοιμάζει - она готовит / собирает
ετοιμάζει - оно готовит / собирает
ετοιμάζουμε - мы готовим / собираем
ετοιμάζετε - вы готовите / собираете
ετοιμάζουν - они готовят / собирают

θα ετοιμάσω - я приготовлю / соберу
θα ετοιμάσεις - ты приготовишь / соберёшь
θα ετοιμάσει - он приготовит / соберёт
θα ετοιμάσει - она приготовит / соберёт
θα ετοιμάσει - оно приготовит / соберёт
θα ετοιμάσουμε - мы приготовим / соберём
θα ετοιμάσετε - вы приготовите / соберёте
θα ετοιμάσουν - они приготовят / соберут
# Контекст
- **Ετοιμάζω το πρωινό.** — Я готовлю завтрак.
- **Ετοιμάσαμε το δείπνο μαζί.** — Мы вместе приготовили ужин.
