# База
дарю / жертвую : δώρισα - δωρίζω - θα δωρίσω

# Тип
Τύπος Α

# Формы
δώρισα - я подарил / пожертвовал
δωρίζω - я дарю / жертвую
δωρίζεις - ты даришь
δωρίζει - он/она дарит
δωρίζουμε - мы дарим
δωρίζετε - вы дарите
δωρίζουν - они дарят
θα δωρίσω - я подарю / пожертвую

# Контекст
- **Δωρίζω ένα βιβλίο στη φίλη μου.** — Я дарю книгу подруге.
- **Δωρίζουν χρήματα στο νοσοκομείο.** — Они жертвуют деньги больнице.
