# База
не в настроении для урока : Δεν έχω κέφι για μάθημα

# Формы
Δεν - не
έχω - имею
κέφι - настроение
για - для
μάθημα - урок
Δεν έχω κέφι για μάθημα - не в настроении для урока

# Контекст
- **Δεν έχω κέφι για μάθημα** — Сегодня не в настроении для урока..
