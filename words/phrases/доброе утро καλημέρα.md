# База
доброе утро : Καλημέρα

# Формы
Καλημέρα - доброе утро
Καλησπέρα - добрый вечер
Καληνύχτα - спокойной ночи

# Контекст
- **Καλημέρα! Πώς είσαι;** — Доброе утро! Как ты?
- **Καλησπέρα, κύριε Νίκο.** — Добрый вечер, господин Никос.
