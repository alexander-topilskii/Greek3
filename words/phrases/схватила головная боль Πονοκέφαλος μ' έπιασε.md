# База
схватила головная боль : Πονοκέφαλος μ' έπιασε

# Формы
Πονοκέφαλος - головная боль
μ' - меня
έπιασε - схватила; настигла
Πονοκέφαλος μ' έπιασε - схватила головная боль

# Контекст
- **Πονοκέφαλος μ' έπιασε** — Уф! Схватила головная боль..
