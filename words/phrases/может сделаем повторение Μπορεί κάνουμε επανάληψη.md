# База
может, сделаем повторение : Μπορεί κάνουμε επανάληψη

# Формы
Μπορεί - может
κάνουμε - мы делаем
επανάληψη - повторение
Μπορεί κάνουμε επανάληψη - может, сделаем повторение

# Контекст
- **Μπορεί κάνουμε επανάληψη** — Может, сначала повторим материал?.
