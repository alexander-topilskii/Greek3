# База
рубашка (одна) : ένα πουκάμισο

# Формы
ένα - один, одна (с.р.)
πουκάμισο - рубашка
ένα πουκάμισο - рубашка (одна)

# Контекст
- **Φόρεσε ένα καθαρό πουκάμισο.** — Надел чистую рубашку.
- **Θέλω να αγοράσω ένα καινούριο πουκάμισο.** — Хочу купить новую рубашку.
