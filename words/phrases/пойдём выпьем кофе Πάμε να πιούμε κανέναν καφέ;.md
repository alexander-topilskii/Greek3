# База
пойдём выпьем кофе? : Πάμε να πιούμε κανέναν καφέ;

# Формы
Πάμε - пойдём
να - (частица инф.)
πιούμε - мы выпьем
κανέναν - какой-нибудь
καφέ - кофе
Πάμε να πιούμε κανέναν καφέ; - пойдём выпьем кофе?

# Контекст
- **Πάμε να πιούμε κανέναν καφέ;** — Пойдём выпьем кофе на перемене?.
