# База
бронировать / делать бронь : κάνω κράτηση

# Формы
κάνω - я делаю; бронирую
κράτηση - бронь
κάνω κράτηση - бронировать; делать бронь

# Контекст
- **Θέλω να κάνω κράτηση για δύο θέσεις.** — Хочу забронировать два места.
- **Κάναμε κράτηση για την παράσταση.** — Забронировали места на спектакль.
