# База
нашла/нашёл работу : Βρήκε δουλειά

# Формы
Βρήκε - он/она нашёл(а)
δουλειά - работу
Βρήκε δουλειά - он/она нашёл(а) работу


# Контекст
- **Βρήκε δουλειά σε μια εταιρεία.** — Он нашёл работу в компании.
- **Η αδελφή μου βρήκε δουλειά τον Ιούνιο.** — Моя сестра нашла работу в июне.
