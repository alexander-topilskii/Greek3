# База
прошу разрешение / отпрашиваюсь : Ζητάω άδεια

# Формы
ζητάω - я прошу
άδεια - разрешение; отпуск
Ζητάω άδεια - прошу разрешение; отпрашиваюсь

# Контекст
- **Ζητάω άδεια από τον διευθυντή.** — Я отпрашиваюсь у директора.
- **Πρέπει να ζητάω άδεια για να φύγω νωρίς.** — Мне нужно отпрашиваться, чтобы уйти пораньше.
