# База
нужно закончить домашнее задание : πρέπει να τελειώσουμε εργασία

# Формы
πρέπει - нужно; должны
να - (частица инф.)
τελειώσουμε - мы закончим
εργασία - домашнее задание
πρέπει να τελειώσουμε εργασία - нужно закончить домашнее задание

# Контекст
- **πρέπει να τελειώσουμε εργασία** — Нужно закончить домашку до завтра..
