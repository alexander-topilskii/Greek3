# База
через 2 недели : σε 2 εβδομάδες

# Формы
σε - через / за
2 - два
εβδομάδες - недели
σε 2 εβδομάδες - через 2 недели

# Контекст
- **Θα φύγουμε σε 2 εβδομάδες.** — Мы уедем через 2 недели.
- **Σε 2 εβδομάδες αρχίζουν οι διακοπές.** — Через 2 недели начинаются каникулы.
