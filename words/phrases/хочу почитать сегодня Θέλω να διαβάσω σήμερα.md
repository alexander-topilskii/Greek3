# База
хочу почитать сегодня : Θέλω να διαβάσω σήμερα

# Формы
Θέλω - я хочу
να - (частица инф.)
διαβάσω - я почитаю
σήμερα - сегодня
Θέλω να διαβάσω σήμερα - хочу почитать сегодня

# Контекст
- **Θέλω να διαβάσω σήμερα** — Сегодня хочу почитать книгу..
