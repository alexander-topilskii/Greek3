# База
его собственные ключи : δικά του κλειδιά


# Формы
δικά - свои / собственные
του - его
κλειδιά - ключи
δικά του κλειδιά - его собственные ключи

# Контекст
- **Τα δικά του κλειδιά είναι στο συρτάρι.** — Его собственные ключи в ящике стола.
- **Μην πάρεις τα δικά του κλειδιά.** — Не бери его собственные ключи.
