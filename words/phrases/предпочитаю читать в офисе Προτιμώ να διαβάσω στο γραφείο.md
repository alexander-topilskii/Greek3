# База
предпочитаю читать в офисе : Προτιμώ να διαβάσω στο γραφείο

# Формы
Προτιμώ - я предпочитаю
να - (частица инф.)
διαβάσω - я почитаю
στο - в
γραφείο - офис
Προτιμώ να διαβάσω στο γραφείο - предпочитаю читать в офисе

# Контекст
- **Προτιμώ να διαβάσω στο γραφείο** — Дома шумно — предпочитаю читать в офисе..
