# База
мы компания из пяти человек с большой любовью к гимнастике : Είμαστε μια παρέα από πέντε άτομα με μεγάλη αγάπη για τη γυμναστική


# Формы
Είμαστε - мы (есть)
μια - одна / компания
παρέα - компания / группа
από - из
πέντε - пять
άτομα - человек(а)
με - с
μεγάλη - большая
αγάπη - любовь
για - для / к
τη - (артикль)
γυμναστική - гимнастика
Είμαστε μια παρέα από πέντε άτομα με μεγάλη αγάπη για τη γυμναστική - мы компания из пяти человек с большой любовью к гимнастике

# Контекст
- **Είμαστε μια παρέα από πέντε άτομα με μεγάλη αγάπη για τη γυμναστική.** — Мы компания из пяти человек с большой любовью к гимнастике.
