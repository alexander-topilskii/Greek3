# База
нужно много читать в эти выходные : Πρέπει να διαβάσω πολύ αυτό το Σαββατοκύριακο

# Формы
Πρέπει - нужно
να - (частица инф.)
διαβάσω - я почитаю
πολύ - много
αυτό - эти; это
το - (артикль)
Σαββατοκύριακο - выходные
Πρέπει να διαβάσω πολύ αυτό το Σαββατοκύριακο - нужно много читать в эти выходные

# Контекст
- **Πρέπει να διαβάσω πολύ αυτό το Σαββατοκύριακο** — В эти выходные нужно много читать..
