# База
счёт, пожалуйста : Τον λογαριασμό, παρακαλώ

# Формы
Τον - (артикль)
λογαριασμό - счёт
παρακαλώ - пожалуйста
Τον λογαριασμό, παρακαλώ - счёт, пожалуйста


# Контекст
- **Τον λογαριασμό, παρακαλώ.** — Счёт, пожалуйста.
- **Μπορώ να έχω τον λογαριασμό;** — Можно счёт?
