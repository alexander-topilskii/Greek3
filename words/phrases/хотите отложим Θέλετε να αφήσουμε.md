# База
хотите (ли), мы оставим / отложим : Θέλετε να αφήσουμε

# Формы
Θέλετε - вы хотите
να - (частица сосл.)
αφήσουμε - мы оставим; отложим
Θέλετε να αφήσουμε - хотите, мы оставим / отложим

# Контекст
- **Θέλετε να αφήσουμε τις εφημερίδες;** — Хотите, мы отложим газеты?
- **Θέλετε να αφήσουμε την ταινία για αργότερα;** — Хотите отложим фильм на потом?
