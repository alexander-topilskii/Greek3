# База
иду читать в библиотеку : Πάω να διαβάσω στη βιβλιοθήκη

# Формы
Πάω - я иду
να - (частица инф.)
διαβάσω - я почитаю
στη - в
βιβλιοθήκη - библиотеку
Πάω να διαβάσω στη βιβλιοθήκη - иду читать в библиотеку

# Контекст
- **Πάω να διαβάσω στη βιβλιοθήκη** — Сейчас иду читать в библиотеку..
