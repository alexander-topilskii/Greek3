# База
могу читать по-гречески : Μπορώ να διαβάσω ελληνικά

# Формы
Μπορώ - я могу
να - (частица инф.)
διαβάσω - я прочитаю; почитаю
ελληνικά - по-гречески
Μπορώ να διαβάσω ελληνικά - могу читать по-гречески

# Контекст
- **Μπορώ να διαβάσω ελληνικά** — Уже могу читать простые тексты по-гречески..
