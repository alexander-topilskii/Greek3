# База
собирает свой чемодан : Ετοιμάζει τη βαλίτσα του


# Формы
Ετοιμάζει - он/она собирает / готовит
τη - (артикль)
βαλίτσα - чемодан
του - его / свой
Ετοιμάζει τη βαλίτσα του - собирает свой чемодан

# Контекст
- **Ετοιμάζει τη βαλίτσα του.** — Он собирает свой чемодан.
