# База
будем есть рыбу : θα φάμε ψάρια


# Формы
θα - (частица будущего времени)
φάμε - будем есть / поедим
ψάρια - рыбу
θα φάμε ψάρια - будем есть рыбу

# Контекст
- **Θα φάμε ψάρια.** — Будем есть рыбу.
- **Αύριο θα φάμε ψάρια στο εστιατόριο.** — Завтра будем есть рыбу в ресторане.
