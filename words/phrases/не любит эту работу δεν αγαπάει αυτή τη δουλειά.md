# База
не любит эту работу : Δεν αγαπάει αυτή τη δουλειά


# Формы
Δεν - не
αγαπάει - он/она любит
αυτή - эту
τη - (артикль)
δουλειά - работу
Δεν αγαπάει αυτή τη δουλειά - не любит эту работу

# Контекст
- **Δεν αγαπάει αυτή τη δουλειά.** — Он/она не любит эту работу.
