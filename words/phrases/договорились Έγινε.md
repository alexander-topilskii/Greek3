# База
договорились! / идёт! : Έγινε

# Формы
Έγινε - договорились!; идёт!; сделано

# Контекст
- **— Πάμε στον θερινό κινηματογράφο; — Έγινε!** — — Пойдём в летний кинотеатр? — Договорились!
- **Έγινε, κάνω κράτηση αμέσως.** — Идёт, бронирую сразу.
