# База
привет (неформ.) : Γεια σου

# Формы
Γεια σου - привет
Γεια σας - здравствуйте (форм.)

# Контекст
- **Γεια σου, Μαρία!** — Привет, Мария!
- **Γεια σου! Τι κάνεις;** — Привет! Как дела?
