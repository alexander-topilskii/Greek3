# База
соревнования по всему миру : αγώνες σε όλο τον κόσμο


# Формы
αγώνες - соревнования
σε - по / в
όλο - весь
τον - (артикль)
κόσμο - мир (вин.)
αγώνες σε όλο τον κόσμο - соревнования по всему миру

# Контекст
- **αγώνες σε όλο τον κόσμο** — соревнования по всему миру (фрагмент: *…αγώνες σε όλο τον κόσμο.*)
