# База
думаю почитать греческую историю : Σκέφτομαι να διαβάσω ελληνική ιστορία

# Формы
Σκέφτομαι - я думаю
να - (частица инф.)
διαβάσω - я почитаю
ελληνική - греческую
ιστορία - историю
Σκέφτομαι να διαβάσω ελληνική ιστορία - думаю почитать греческую историю

# Контекст
- **Σκέφτομαι να διαβάσω ελληνική ιστορία** — Думаю почитать греческую историю на выходных..
