# База
льготный (о билете) : μειωμένος - μειωμένη - μειωμένο

# Формы
μειωμένος - льготный (м.р.)
μειωμένη - льготная (ж.р.)
μειωμένο - льготное; льготный билет (с.р.)

# Контекст
- **Έχουν μειωμένα εισιτήρια για φοιτητές.** — Есть льготные билеты для студентов.
- **Πήρα ένα μειωμένο εισιτήριο.** — Взял льготный билет.
