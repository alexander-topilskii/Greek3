# База
знаменитый : διάσημος - διάσημη - διάσημο

# Формы
διάσημος - знаменитый (м.р.)
διάσημη - знаменитый (ж.р.)
διάσημο - знаменитый (с.р.)

# Контекст
- **Είναι ένας διάσημος ηθοποιός.** — Он знаменитый актёр.
- **Αυτό το εστιατόριο είναι διάσημο για τα ψάρια του.** — Этот ресторан знаменит своей рыбой.
