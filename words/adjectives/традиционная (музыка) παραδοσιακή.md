# База
традиционная (музыка) : παραδοσιακός - παραδοσιακή - παραδοσιακό

# Формы
παραδοσιακός - традиционный (м.р.)
παραδοσιακή - традиционная (ж.р.)
παραδοσιακό - традиционное (с.р.)

# Контекст
- **Ακούμε παραδοσιακή μουσική.** — Слушаем традиционную музыку.
- **Μου αρέσει η παραδοσιακή μουσική.** — Мне нравится традиционная музыка.
