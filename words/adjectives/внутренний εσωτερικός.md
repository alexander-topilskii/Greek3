# База
внутренний : εσωτερικός - εσωτερική - εσωτερικό

# Формы
εσωτερικός - внутренний (м.р.)
εσωτερική - внутренний (ж.р.)
εσωτερικό - внутренний (с.р.)

# Контекст
- **Ο εσωτερικός χώρος είναι μεγάλος.** — Внутреннее пространство большое.
- **Το δωμάτιο έχει εσωτερικό μπάνιο.** — В комнате внутренняя ванная.
