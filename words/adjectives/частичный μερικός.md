# База
частичный / некоторый : μερικός - μερική - μερικό

# Формы
μερικός - частичный / некоторый (м.р.)
μερική - частичная / некоторая (ж.р.)
μερικό - частичное / некоторое (с.р.)

# Контекст
- **Έχουμε μερική επιτυχία.** — У нас частичный успех.
- **Μερικοί άνθρωποι δεν συμφωνούν.** — Некоторые люди не согласны.
