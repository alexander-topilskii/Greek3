# База
ежедневный : καθημερινός - καθημερινή - καθημερινό

# Формы
καθημερινός - ежедневный (м.р.)
καθημερινή - ежедневный (ж.р.)
καθημερινό - ежедневный (с.р.)

# Контекст
- **Ο περίπατος είναι μέρος της καθημερινής μου ρουτίνας.** — Прогулка — часть моего ежедневного распорядка.
- **Είναι μια καθημερινή συνήθεια.** — Это ежедневная привычка.
