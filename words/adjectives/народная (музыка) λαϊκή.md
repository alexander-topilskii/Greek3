# База
народная (музыка) : λαϊκός - λαϊκή - λαϊκό

# Формы
λαϊκός - народный (м.р.)
λαϊκή - народная (ж.р.)
λαϊκό - народное (с.р.)

# Контекст
- **Ακούμε λαϊκή μουσική.** — Слушаем народную музыку.
- **Η λαϊκή μουσική είναι πολύ δημοφιλής.** — Народная музыка очень популярна.

# Примечание
Не путать с **λαϊκή** (λαϊκή αγορά) — рынок, базар.
