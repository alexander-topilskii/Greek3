# База
например : π.χ.

# Тип
επέκταση

# Формы
π.χ. / για παράδειγμα - например

# Контекст
- **Μου αρέσουν τα φρούτα, π.χ. τα μήλα και τα πορτοκάλια.** — Мне нравятся фрукты, например яблоки и апельсины.
- **Μπορείς να πιεις κάτι ζεστό, π.χ. τσάι.** — Ты можешь выпить что-то горячее, например чай.
