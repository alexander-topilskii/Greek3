# База
родительный падеж : γενική - принадлежность - του · της · του

# Тип
γενική

# Формы
του φίλου - друга (м.р., ед., −ος)
του γείτονα - соседа (м.р., ед., −ας)
του φοιτητή - студента (м.р., ед., −ης)
της γυναίκας - женщины (ж.р., ед., −α)
της αδερφής - сестры (ж.р., ед., −η)
του μωρού - младенца (с.р., ед., −ο)
του παιδιού - ребёнка (с.р., ед., −ι)
του διαμερίσματος - квартиры (с.р., ед., −μα)
των φίλων - друзей (м.р., мн.)
των γυναικών - женщин (ж.р., мн.)
των παιδιών - детей (с.р., мн.)
των διαμερισμάτων - квартир (с.р., мн.)

# Назначение
Родительный падеж (γενική) отвечает на вопросы **кого?** и **чего?**

Главные функции:
1. **Принадлежность** — чей? чья? чьё?
2. После предлогов **από** (от), **για** (для), **χωρίς** (без), **μέχρι** (до)

Το βιβλίο του φίλου. — Книга друга.
Το σπίτι του γείτονα. — Дом соседа.
Το δωμάτιο της αδερφής. — Комната сестры.
Το κλειδί του διαμερίσματος. — Ключ от квартиры.

# Образование
**Артикль родительного + особое окончание.**

| Род | Ед. число | Мн. число |
|-----|-----------|-----------|
| м.р. | του …−ου / −α / −ή | των …−ων / −ών |
| ж.р. | της …−ας / −ης | των …−ων / −ών |
| с.р. | του …−ου / −ιού / −ματος | των …−ων / −ιών / −μάτων |

Типичные изменения (ед. → род.):
- φίλος → **του φίλου** (−ος → −ου)
- γείτονας → **του γείτονα** (−ας → −α)
- φοιτητής → **του φοιτητή** (−ης → −ή)
- γυναίκα → **της γυναίκας** (−α → −ας)
- αδερφή → **της αδερφής** (−η → −ης)
- μωρό → **του μωρού** (−ο → −ου)
- παιδί → **του παιδιού** (−ι → −ιού)
- διαμέρισμα → **του διαμερίσματος** (−μα → −ματος)

Во мн. числе артикль **των** для всех родов; ударение часто переходит на **−ών**:
- φίλοι → **των φίλων**, γείτονες → **των γειτόνων**, γυναίκες → **των γυναικών**
- παιδιά → **των παιδιών**, διαμερίσματα → **των διαμερισμάτων**

**В разговорной речи** часто вместо падежной формы говорят:
το βιβλίο **του φίλου μου** — букв. «книга моего друга» (μου = мой, род. местоимение).

# Примеры
Ο καφές της μαμάς. — Кофе мамы.
Το όνομα του παιδιού. — Имя ребёнка.
Τα σπίτια των γειτόνων. — Дома соседей.
Έρχομαι από το σχολείο. — Я прихожу из школы.
Χωρίς νερό. — Без воды.
