# База
почтовая марка : το γραμματόσημο - τα γραμματόσημα

# Тип
ουδέτερο

# Формы
το γραμματόσημο - почтовая марка (ед.)
τα γραμματόσημα - почтовые марки (мн.)

# Контекст
- **Αγόρασα γραμματόσημα στο ταχυδρομείο.** — Купил почтовые марки на почте.
- **Βάλε δύο γραμματόσημα στον φάκελο.** — Наклей два марки на конверт.
