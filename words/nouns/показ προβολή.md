# База
показ / сеанс : η προβολή - οι προβολές

# Тип
θηλυκό

# Формы
η προβολή - показ, сеанс (ед.)
οι προβολές - показы, сеансы (мн.)

# Контекст
- **Η προβολή ξεκινά στις εννιά.** — Сеанс начинается в девять.
- **Ποια είναι η επόμενη προβολή;** — Какой следующий сеанс?
