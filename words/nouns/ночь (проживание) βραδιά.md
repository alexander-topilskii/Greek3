# База
ночь (проживание) / вечер : η βραδιά - οι βραδιές

# Тип
θηλυκό

# Формы
η βραδιά - ночь проживания / вечер (ед.)
οι βραδιές - ночь проживания / вечер (мн.)

# Контекст
- **Η διαμονή κοστίζει 80 ευρώ ανά βραδιά.** — Проживание стоит 80 евро за ночь.
- **Κλείσαμε δύο βραδιές στο ξενοδοχείο.** — Мы забронировали две ночи в отеле.
