# База
домашнее задание / работа : η εργασία - οι εργασίες

# Тип
θηλυκό

# Формы
η εργασία - домашнее задание; работа (ед.)
οι εργασίες - задания; работы (мн.)

# Контекст
- **Πρέπει να τελειώσουμε την εργασία.** — Нужно закончить домашнее задание.
- **Η εργασία είναι δύσκολη.** — Задание сложное.
