# База
ресторан : το εστιατόριο - τα εστιατόρια

# Тип
ουδέτερο

# Формы
το εστιατόριο - ресторан (ед.)
τα εστιατόρια - ресторан (мн.)

# Контекст
- **Το εστιατόριο σερβίρει καλό φαγητό.** — Ресторан подаёт хорошую еду.
- **Καθίσαμε σε ένα εστιατόριο δίπλα στη θάλασσα.** — Мы сели в ресторан у моря.
