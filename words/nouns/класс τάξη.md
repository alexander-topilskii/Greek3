# База
класс (школьный) : η τάξη - οι τάξεις

# Тип
θηλυκό

# Формы
η τάξη - класс (ед.)
οι τάξεις - классы (мн.)
Με ποια τάξη; - с каким классом?

# Контекст
- **Με ποια τάξη πηγαίνετε στο σχολείο;** — С каким классом вы ходите в школу?
- **Η τάξη μας έχει είκοσι πέντε μαθητές.** — В нашем классе двадцать пять учеников.
