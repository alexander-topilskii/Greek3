# База
газета : η εφημερίδα - οι εφημερίδες

# Тип
θηλυκό

# Формы
η εφημερίδα - газета (ед.)
οι εφημερίδες - газеты (мн.)

# Контекст
- **Διαβάζω την εφημερίδα κάθε πρωί.** — Я читаю газету каждое утро.
- **Αγόρασα δύο εφημερίδες.** — Купил две газеты.
