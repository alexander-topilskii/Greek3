# База
приключение / приключенческий фильм : η περιπέτεια - οι περιπέτειες

# Тип
θηλυκό

# Формы
η περιπέτεια - приключение; приключенческий фильм (ед.)
οι περιπέτειες - приключения; приключенческие фильмы (мн.)

# Контекст
- **Μου αρέσουν οι περιπέτειες.** — Мне нравятся приключенческие фильмы.
- **Είναι μια συναρπαστική περιπέτεια.** — Это захватывающий приключенческий фильм.
