# База
пекарь : ο φούρναρης - οι φουρναρηδες

# Тип
αρσενικό

# Формы
ο φούρναρης - пекарь (ед.)
οι φουρναρηδες - пекарь (мн.)

# Контекст
- **Ο φούρναρης φτιάχνει φρέσκο ψωμί κάθε πρωί.** — Пекарь каждое утро печёт свежий хлеб.
- **Ο φούρναρης της γειτονιάς είναι πολύ ευγενικός.** — Пекарь из нашего района очень приветливый.
