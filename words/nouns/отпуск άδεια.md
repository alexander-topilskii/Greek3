# База
отпуск : η άδεια - οι άδειες

# Тип
θηλυκό

# Формы
η άδεια - отпуск (ед.)
οι άδειες - отпуска (мн.)
Θα πάρει άδεια - он/она возьмёт отпуск

# Контекст
- **Θα πάρει άδεια τον Αύγουστο.** — Он возьмёт отпуск в августе.
- **Η συνάδελφός μου θα πάρει άδεια δύο εβδομάδες.** — Моя коллега возьмёт отпуск на две недели.
