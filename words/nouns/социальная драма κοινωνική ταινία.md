# База
социальная драма / фильм на социальную тему : η κοινωνική ταινία - οι κοινωνικές ταινίες

# Тип
θηλυκό

# Формы
η κοινωνική ταινία - социальная драма (ед.)
οι κοινωνικές ταινίες - социальные драмы (мн.)

# Контекст
- **Η ταινία είναι μια κοινωνική ταινία.** — Фильм — социальная драма.
- **Βλέπουμε μια κοινωνική ταινία.** — Мы смотрим фильм на социальную тему.
