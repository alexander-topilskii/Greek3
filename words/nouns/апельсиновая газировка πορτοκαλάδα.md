# База
апельсиновая газировка : η πορτοκαλάδα - οι πορτοκαλάδες

# Тип
θηλυκό

# Формы
η πορτοκαλάδα - апельсиновая газировка (ед.)
οι πορτοκαλάδες - апельсиновые газировки (мн.)

# Контекст
- **Πήρα μια πορτοκαλάδα από το μαγαζί.** — Я взял апельсиновую газировку в магазине.
- **Η πορτοκαλάδα είναι παγωμένη.** — Апельсиновая газировка холодная.
