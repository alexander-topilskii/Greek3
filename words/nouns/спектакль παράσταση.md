# База
спектакль / представление : η παράσταση - οι παραστάσεις

# Тип
θηλυκό

# Формы
η παράσταση - спектакль, представление (ед.)
οι παραστάσεις - спектакли, представления (мн.)

# Контекст
- **Πηγαίνουμε στην παράσταση.** — Мы идём на спектакль.
- **Η παράσταση ξεκινά στις οκτώ.** — Представление начинается в восемь.
